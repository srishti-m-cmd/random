# random 🗂️

A sandbox repo for quick tests, experiments, and random files I don't want cluttering my main projects.

## What's in here

- `del.txt` / `fake.txt` — throwaway test files
- `names.txt` — random data/sample file

## Why this exists

Sometimes you just need a scratch space on GitHub — for testing Git commands, trying out workflows, or dumping files without worrying about structure. That's this repo.
