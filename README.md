## 🤔 what even is this

Not every idea deserves its own repo. Not every Git command deserves to be tested on a real project. This is where that stuff goes instead — no structure, no judgment, no cleanup required.

> tl;dr — this is my digital junk drawer, and it's thriving 🧃

Nothing here is precious. Everything here is disposable. That's the whole point.

## 🎯 why this exists

Sometimes you just need a scratch space —
- testing a sketchy Git command before running it somewhere that matters 😅
- trying out a new workflow without fear of breaking prod (there is no prod)
- dumping random files without overthinking folder structure
- practicing `git` merges, branches, and PRs risk-free

That's this repo. Low stakes, high freedom.
